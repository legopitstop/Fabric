# [<](../README.md) Canned

![alt](icon.png)

This mod adds canned foods! You can place these blocks on the ground. You can add up to 8 cans in one block.

## Mod Data

| Key         | Value      |
|-------------|------------|
| supported   | `yes`      |
| version     | `1.0.0`    |
| mc_version  | `1.20.x`   |
| mod_id      | `canned`   |
| client_side | `required` |
| server_side | `required` |
