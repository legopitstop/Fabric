# [<](../README.md) Basalt Blocks

![alt](icon.png)

Two basalt blocks, not enough? Not worth the time to gather? This mod adds basalt blocks to further decorate your world, house, or nether castle!

| Key         | Value          |
|-------------|----------------|

| version     | `1.0.2`        |
| mc_version  | `1.21`         |
| mod_id      | `basaltblocks` |

| server_side | `required`     |
